## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- [Python 3](https://www.python.org/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/raymondkneipp/wizard-game.git
   ```
2. Play game
   ```sh
   python3 main.py
   ```
